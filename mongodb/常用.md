对两个数求和然后排序
```
db.getCollection('sales').aggregate([
{
    $project: {
      item: 1,
      price: 1,
      quantity: 1,
      date: 1,  
      totalSum: { $sum: { $sum: [ "$price", "$quantity" ] }}
   }
},
{
    $sort: {totalSum: 1}
}
])
```
